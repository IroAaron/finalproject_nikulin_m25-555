# Инструкция по запуску трейдингого сервиса:

1) Установка зависимостей: make install
2) Запуск программы: make project
3) Сборка проекта: make build
4) Публикация: make publish
5) Установка всех необходимых пакетов: make package-install
6) Проверка кода в соответствии с ruff: make line

# Инструкция по работе с приложением:

register - регистрация пользователя (при регистрации выставляется стандартное значение счета: 1 000 000 USD)
login - вход в аккаунт
portfolio - просмотр финансового портфеля
buy - покупка валюты
sell - продажа валюты
get-rate - просмотр списка курсов валют
show-rates - просмотр списка курсов валют (с фильтрацией)
update - обновление списка валют
parser - состояние парсера
autoupdate - запуск автоматического обновления парсера
stop - остановка автоматического обновления парсера
exit - выход из приложения
